# textual-experiments

A sandpit repo for testing [textual](https://github.com/willmcgugan) related things.

## Setup

``` bash
poetry install
pre-commit install
```

## autocomplete

``` bash
python -m src.autocomplete
```

## Button widget

``` bash
python -m src.button_widget
```

## Figlet widget

``` bash
python -m src.figlet_widget
```

## Single stat widget

``` bash
python -m src.single_stat
```
